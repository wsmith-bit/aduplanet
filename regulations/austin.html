Austin ADU — at-a-glance

Zoning
Classic ADUs allowed in SF-1, SF-2, SF-3 contexts; check base district + overlays. {{verify locally}}
Minimum lot area
≥ 5,750 sq ft for the ADU (Two-Family Residential) path. Smaller lots may add a primary unit via HOME small-lot rules (not an ADU). {{verify locally}}
Parking
Citywide minimum off-street parking eliminated (Nov 2, 2023). Site constraints/accessible spaces still apply.
Owner-occupancy
Not required by city ADU regulations (historic no owner-occ). Confirm HOA/loan rules. {{verify locally}}
STR use
ADUs built after Oct 1, 2015 generally capped at 30 STR nights/year with license. {{verify locally}}
Height/size
Follows base zone + objective standards; confirm current code for height, FAR, separation, and compatibility. {{verify locally}}

The HOME Amendments create additional ways to add units (e.g., small-lot single-family); those are distinct from the “ADU” (Two-Family Residential) path summarized here.

How to permit an ADU in Austin (5 steps)

  1. Verify eligibility: Use the Property Profile to confirm zoning, overlays, and lot area; review the City’s ADU page and HOME Amendments. {{verify locally}}
  2. Get an address: Request a new ADU address/building number via Address Management (required before applying).
  3. Set up AB+C: Create/confirm your Austin Build + Connect account (applicant email must match permit record).
  4. Apply online: Start the Residential Plan Review application from the City’s ADU page; upload plans through ProjectDox.
  5. Reviews, permits, inspections: Respond to comments, pay fees, pull permits, schedule inspections, and obtain the Certificate of Occupancy.

FAQ — Austin ADUs

Does my lot qualify if it’s under 5,750 sq ft?

Not for the classic ADU path. However, the HOME Amendments created a small-lot single-family path for one primary unit on lots as small as ~1,800 sq ft in certain zones. That path is not an ADU. {{verify locally}}

How many units can I have total?

Unit counts depend on base zoning and HOME allowances (e.g., duplex/three-unit allowances in certain contexts). Check the HOME page and your parcel’s zoning report. {{verify locally}}

Do I need extra parking for the ADU?

No citywide minimum off-street parking since Nov 2023, unless an overlay or specific requirement applies. Accessible spaces still apply. {{verify locally}}

Can I short-term rent the ADU?

ADUs constructed after Oct 1, 2015 are typically limited to 30 STR nights/year and require STR licensing. Review the City’s STR program for details. {{verify locally}}

Where should I start?

Run your parcel in Property Profile, scan the City ADU page, then line up Costs, Builders, and Financing. Apply via AB+C and upload through ProjectDox.
